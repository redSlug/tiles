# Tiles

[try it out!](https://redslug.github.io/tiles/)

![game.png](game.png)

## Setup

Install and run game

```
nvm install 22
cd tiles
npm install
npm run dev
```

## Deploy

### Option 1:

[Use github Pages](.github/workflows/deploy.yaml)

### Option 2:

```
npm run deploy
```
